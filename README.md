This Github repository was created by Aarav Rastogi to save the work produced during a summer internship at the Indian Institute of Technology Bombay (IITB), conducted during the summer of 2026. It accompanies a guide to image analysis and processing, which is the file labelled 'A Note on Astronomical Image Processing and Analysis'. 

This analysis is based on data colleccted by the GROWTH India Telescope (GIT) of GRB 230204B. The afterglow discovery paper can be found here: https://arxiv.org/abs/2512.03136

Intructions for using this repository:
This repository is free to use. The zero-point calaculations are explained in the note, and use a radius of 9px. The file labelled 'GRB 230204B Plots (4).ipynb' contains the code for plotting the afterglow's lightcurve.
